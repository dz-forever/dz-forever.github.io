# **git配置ssh**

### 1.配置用户名和邮箱

```
git config --global user.name "用户名"
git config --global user.email "邮箱"
```

### 2.生成密钥

```
ssh-keygen -t rsa
```

### 3.找到公钥并添加到git仓库

```
cd ~/.ssh
vi id_rsa.pub
```

将id_rsa.pub里的公钥添加到git仓库。
